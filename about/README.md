# О приложении

**BarberShop** — мобильное приложение для записи в барбершоп.

> Создаем стиль, который говорит о вас.

## Пример кода

```python
class BarberShop:
    def __init__(self, name, location):
        self.name = name
        self.location = location
    
    def book_service(self, service_type):
        return f"Запись на {service_type} в {self.name}"
```
