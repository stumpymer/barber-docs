# Частые вопросы

## Запись и услуги

| Вопрос | Ответ |
|--------|-------|
| Какие услуги предоставляются? | Стрижка, бритье, укладка, оформление бороды |
| Можно ли отменить запись? | Да, отмена возможна за 2 часа до визита |
| Какие способы оплаты? | Наличные, карта, онлайн-оплата |

## Пример кода для работы с API

```javascript
// Получение доступных слотов для записи
async function getAvailableSlots(barberId, date) {
  const response = await fetch(`/api/slots/${barberId}?date=${date}`);
  return await response.json();
}
```
